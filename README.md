## About The Project

[Read the Blog Post](https://www.haylium.com/the-quest-for-shoes/)

I love five fingers and have been wearing them for years.
They are pricer than what I would like, especially considering how quickly I run through pairs.
I like to buy more than one at a time so I often try to buy pairs that are on sale. As of late
it has gotten much harder for me to find pairs on sale in my size, so I decided to create a script that
checks their website for shoes in my size and then sends me a text message.
